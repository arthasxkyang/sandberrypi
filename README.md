# sandberrypi
---
## 如何使用
### 安装Python
需要python3.11以上版本
### 下载代码 
```shell
git clone https://github.com/arthasxkyang/sandberrypi.git
```
### 安装依赖
```shell
cd sandberrypi
pip install -r requirements.txt
```
### 运行代码
```shell
python run.py
```
### 访问网页
在浏览器中输入`http://localhost:5000`即可访问
默认的首页有主要函数和url的说明
